# تحديث مهم

تم تحويل المشروع من تقرير نظري إلى تنفيذ Backend عملي واحترافي.

## المتاح الآن

- API server فعلي عبر FastAPI.
- تحليل سوق + تحليل جمهور.
- تخطيط استراتيجية.
- Autopilot لتوليد وجدولة محتوى تلقائياً.
- نشر تلقائي على Facebook وسحب Insights.
- Retry logic للنشر عند أخطاء الشبكة.
- نشر فوري لأي منشور عبر endpoint.
- تتبع محاولات النشر والأخطاء لكل منشور.
- ملخص KPI عبر endpoint مخصص.

## تطوير إضافي مقترح

- A/B Testing تلقائي لرفع الأداء.
- Queue احترافي (Redis/Celery) للتوسع.
- Multi-tenant + صلاحيات.
- Dashboard أمامية لإدارة الأداء.

## نقطة البداية

راجع `README.md` لمعرفة خطوات إضافة المفاتيح وتشغيل السيرفر وتنفيذ Autopilot.
